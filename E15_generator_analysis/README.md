# E15 – AC Generator Data Analysis

This project will contain a Python-based reanalysis of the E15 experiment on an AC generator.

## Goal
To reproduce and analyze the main experimental relationships from the laboratory experiment, including:
- induced voltage vs angular velocity
- terminal voltage vs angular velocity for different load resistances
- terminal voltage vs load resistance
- power transfer vs load resistance

## Skills developed
- Python data handling
- scientific plotting
- comparison of theory and experiment
- interpretation of experimental measurements
- physics-based analysis of generator behaviour
